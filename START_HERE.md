# 📱 View SwiftDrop on Your Phone - Quick Start

## ✅ Setup Complete!

All files are ready. Follow these simple steps:

---

## 🚀 Steps to View on Your Phone

### 1️⃣ On Your Computer (in Cursor):

Open the terminal in Cursor (press **Ctrl + `** backtick key)

Then type:
```bash
cd /home/immigrant/dropoff
npx expo start
```

Wait for the QR code to appear in the terminal.

---

### 2️⃣ On Your Phone:

**Download Expo Go:**
- **iPhone:** App Store → Search "Expo Go" → Install (FREE)
- **Android:** Play Store → Search "Expo Go" → Install (FREE)

**Open the App:**
- Open Expo Go on your phone
- **iPhone:** Scan QR code with camera
- **Android:** Tap "Scan QR Code" in Expo Go, then scan

---

### 3️⃣ That's It! 🎉

Your SwiftDrop app will load on your phone instantly!

Every time you save a file in Cursor, the app updates automatically on your phone.

---

## 🎯 What You'll See

The app starts with the **Customer Onboarding** screen. You can navigate through:

- **Customer App:** Onboarding → Login → Home → Order Flow
- **Driver App:** Navigate to driver screens
- **Admin Dashboard:** View admin panels

---

## 🔧 Troubleshooting

**QR code not appearing?**
- Make sure npm install finished
- Try: `npm install` then `npx expo start`

**Phone can't scan?**
- Make sure phone and computer are on same WiFi
- Try typing the URL shown in terminal into Expo Go

**App crashes?**
- Some screens might need minor adjustments
- Check terminal for error messages
